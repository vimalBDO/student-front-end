## ?? 3-Step Quick Start

### Step 1: Create & Setup
```bash
ng new student-app-angular
cd student-app-angular
npm install
```

### Step 2: Copy Files
Copy all provided application files into the `src/` directory

### Step 3: Run
```bash
# Configure API URL if needed (default: localhost:5000)
ng serve
# Open http://localhost:4200
```
